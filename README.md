Clean GitHub / GitHub Pages package for the MannyPay website.

Upload the contents of this folder to a GitHub repository.

## GitHub Pages settings

- Source: Deploy from a branch

## Do not remove

- `studio-baked-state.js`
- `studio-baked-overrides.css`
- `studio-state.json`
- `assets/`

These files keep the current visual design/edit state intact.
